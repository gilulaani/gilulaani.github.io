@gill/gillkits

Powerful Minecraft Kit Plugin

v1.0.0 Minecraft Plugin

Overview

@gill/gillkits is a high-performance Minecraft plugin that provides advanced kit management capabilities. Built with modern Java practices, it offers:

  • Lightning Fast: Optimized for performance with minimal server impact
  • Easy Configuration: Simple YAML-based configuration
  • Advanced Features: Cooldowns, permissions, custom messages
  • Multi-World: Support for multiple worlds and servers
Download v1.0.0

Installation

Prerequisites

  • Minecraft Server 1.21+
  • Java 11 or higher
  • Spigot/Paper server

Quick Start

  1. Download the latest JAR from the button above
  2. Place it in your plugins/ folder
  3. Restart your server
  4. Configure plugins/GillKits/config.yml
Note: First startup will generate default configuration files.

Configuration

The main configuration file is located at plugins/GillKits/config.yml:

# GillKits Configuration
settings:
  # Kit cooldown in seconds (0 = disabled)
  cooldown: 3600
  
  # Whether to save kit data on player quit
  save-on-quit: true
  
  # Default kit permission node
  default-permission: "gillkits.use"

# Kit definitions
kits:
  starter:
    name: "Starter Kit"
    cooldown: 1800
    items:
      - "DIAMOND_SWORD:1"
      - "DIAMOND_PICKAXE:1"
      - "DIAMOND_HELMET:1"
      - "DIAMOND_CHESTPLATE:1"
      - "DIAMOND_LEGGINGS:1"
      - "DIAMOND_BOOTS:1"
    
  vip:
    name: "VIP Kit"
    cooldown: 7200
    permission: "gillkits.vip"
    items:
      - "NETHERITE_SWORD:1"
      - "NETHERITE_PICKAXE:1"
      - "NETHERITE_HELMET:1"
      - "NETHERITE_CHESTPLATE:1"
      - "NETHERITE_LEGGINGS:1"
      - "NETHERITE_BOOTS:1"

Commands

/gkits create [name]

Create a new kit with the specified name.

Permission: gillkits.admin

/gkits remove [name]

Remove an existing kit.

Permission: gillkits.admin

/gkits give [kit] [player]

Give a kit to a player (bypasses cooldown).

Permission: gillkits.admin

/gkits edit [name]

Edit an existing kit's configuration.

Permission: gillkits.admin

/gkits list

List all available kits.

Permission: gillkits.list

/gkits reload

Reload configuration files.

Permission: gillkits.admin

/gkits help

Show help information and available commands.

Permission: gillkits.help

Examples

Custom Kit with Commands

kits:
  custom:
    name: "Custom Kit"
    cooldown: 86400  # 24 hours
    permission: "gillkits.custom"
    items:
      - "DIAMOND_SWORD:1"
      - "DIAMOND_PICKAXE:1"
    commands:
      - "say %player% claimed the custom kit!"
      - "effect give %player% speed 30 1"

Kit with Enchanted Items

kits:
  enchanted:
    name: "Enchanted Kit"
    items:
      - "DIAMOND_SWORD:1:unbreaking:3,sharpness:5"
      - "DIAMOND_PICKAXE:1:efficiency:5,fortune:3"
