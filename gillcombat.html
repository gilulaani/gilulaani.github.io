@gill/gillcombat

Powerful Minecraft Combat Plugin

v1.0.0 Minecraft Plugin

Overview

@gill/gillcombat is a robust Minecraft plugin that prevents combat logging, blocks commands during combat, and supports AFK detection, WorldGuard, and more. Perfect for PvP servers that want fair fights and no interruptions.

  • Combat Tagging: Prevents players from escaping combat unfairly
  • Command Blocking: Disables specific commands while tagged
  • World & Region Support: Disable tagging in certain worlds/regions
  • AFK Detection: Marks inactive players and prevents abuse
  • Customizable Messages: Full color code/message support
  • Action Bar Countdown: Visual combat timer for players
Download v1.0.0

Installation

Prerequisites

  • Minecraft Server 1.21+
  • Java 11 or higher
  • Spigot/Paper server

Quick Start

  1. Download the latest JAR from the button above
  2. Place it in your plugins/ folder
  3. Restart your server
  4. Configure plugins/GillCombat/config.yml
Note: First startup will generate default configuration files.

Configuration

The main configuration file is located at plugins/GillCombat/config.yml:

# GillCombat Configuration
# List of worlds where tagging is disabled
disabled-worlds: 
  - test
					
# List of WorldGuard region names where tagging is disabled
disabled-regions:
  - test
					
# List of commands that tagged players cannot use
disabled-commands:
  - /spawn
  - /hub
  - /lobby
					
# Duration (in seconds) a player remains tagged after being hit
tag-duration: 15
					
# Time (in seconds) of inactivity before a player is marked AFK
afk-timeout-seconds: 5
					
# Enable debug logging
debug: false
					
# Enable action bar countdown display
enable-action-bar: true
					
# Action bar update frequency (in ticks, 5 = 0.25 seconds)
action-bar-update-frequency: 5

Commands

/gcombat reload

Reload the plugin configuration.

/gcombat disableworld [world]

Disable combat tagging in a specific world.

/gcombat enableworld [world]

Enable combat tagging in a specific world.

/gcombat untag [player]

Remove combat tag from a player.

/gcombat list

List all currently tagged players.

/gcombat worldguard

Check WorldGuard status.

/tag

Check your combat status.

/combat

Check your combat status.

/combatpartners

Show who you are currently in combat with.
